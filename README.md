## Features

- Responsive design
- Modern UI
- App Router
- SEO metadata
- Reusable components
- Optimized images
- Clean component architecture

## Tech Stack

- Next.js 15
- React
- TypeScript
- Tailwind CSS

## Getting Started

```bash
cd frontend/
pnpm install
pnpm run dev
```
